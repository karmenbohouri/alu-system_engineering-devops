# alu-system_engineering-devops

This project is part of the DevOps curriculum. The goal is to learn how to configure redundant web servers and set up a load balancer to improve the availability and reliability of the infrastructure.

## Concepts Covered
* Server configuration automation with Bash.
* Configuring custom HTTP headers with Nginx.
* Installing and configuring HAProxy.
* Network traffic management and load balancing.
